# **Virtual Portfolio Analyzer**

# ===================================

# A serverless full-stack stock portfolio simulator built on AWS.

# Users get ₹10,000 virtual funds and can buy/sell real stocks

# using live market prices — no real money involved.

# 

# **Status**

# **Currently in development**

# 

# Tech Stack

# \- Frontend: React, hosted on S3 + CloudFront

# \- Auth: Amazon Cognito

# \- API: Amazon API Gateway + AWS Lambda (Python 3.13)

# \- Database: Amazon DynamoDB (3 tables)

# \- Market Data: Alpha Vantage API

# \- Infrastructure: AWS IAM, CloudWatch, SQS

# 

# **Architecture**

# &#x20; Features

# \- Virtual wallet with ₹10,000 starting balance

# \- Buy and sell stocks with live prices

# \- Portfolio dashboard with P\&L tracking

# \- Analytics — best/worst performer, allocation

# \- Smart insights — concentration warnings,

# &#x20; diversification suggestions

# \- Complete transaction history
